# 📸 Chrome Web Store 截图制作指南

## 📐 技术要求

### 尺寸规格
Chrome Web Store要求：
- **推荐尺寸：** 1280 x 800 像素
- **最小尺寸：** 640 x 400 像素
- **宽高比：** 16:10 或 8:5
- **格式：** PNG 或 JPEG
- **数量：** 最少1张，最多5张

## 🖼️ 截图方法

### 方法1：使用Chrome开发者工具（推荐）

1. **打开扩展弹窗**
2. **右键点击扩展弹窗** → 选择"检查"
3. **在开发者工具中：**
   - 点击"Toggle device toolbar"图标（📱）
   - 设置自定义尺寸：宽度 1280，高度 800
4. **截图：**
   - Windows: Win + Shift + S
   - Mac: Cmd + Shift + 4
5. **保存为PNG格式**

### 方法2：使用截图工具

**Windows：**
- Win + Shift + S（截图工具）
- 或使用Snagit、ShareX等专业工具

**Mac：**
- Cmd + Shift + 4（选择区域截图）
- Cmd + Shift + 5（截图工具栏）

### 方法3：使用Chrome扩展

安装截图扩展，如：
- Awesome Screenshot
- Nimbus Screenshot
- FireShot
