# 第一部分：免费推广策略

## 一、搜索引擎提交（必做基础）

### 1. Bing提交

**操作步骤：**

1. 访问 [Bing Webmaster Tools](https://www.bing.com/webmasters)
2. 用Microsoft账号登录
3. 添加网站并验证所有权（HTML标签/DNS/文件上传）
4. 提交sitemap（如果已在Google Search Console验证，可直接同步导入）
5. **重要：** 提交Bing = 同时提交Yahoo（Yahoo使用Bing技术）

**注意事项：**
- 提交后1-2周开始收录
- 定期检查索引状态和错误报告

---

### 2. 百度提交

**操作步骤：**

1. 访问 [百度搜索资源平台](https://ziyuan.baidu.com)
2. 注册百度账号并登录
3. 添加网站并完成验证
4. 提交sitemap（仅验证后可用）
5. 使用"链接提交工具"手动提交重要页面

**注意事项：**
- 如果网站有中文内容，收录效果更好
- 百度对内容审查严格，确保内容合规
- 考虑服务器在中国境内，收录速度更快（但非必须）

# 第二部分：自动化工具方案

## ⚠️ 重要风险警告

### 自动化操作的风险：

1. **账号封禁** - 大部分平台（Reddit、LinkedIn、Twitter等）明确禁止自动化操作
2. **Google惩罚** - 批量发外链、自动评论等会被判定为spam
3. **IP封锁** - 短时间大量操作会触发反spam机制
4. **法律风险** - 某些自动化操作可能违反服务条款

### 建议原则：

- ✅ 使用自动化工具**辅助**而非完全自动
- ✅ 保持"半自动"模式（AI生成+人工审核）
- ✅ 限制频率，模拟人类行为
- ✅ 优先使用官方API而非爬虫

---

## 二、AutoCommentAI Chrome扩展

### 功能概述

半自动化的AI评论生成器，支持博客和论坛评论+外链植入。

**Chrome商店评分：** 4.0/5 (753评分)

### 主要功能

1. ✅ **智能内容提取** - 自动识别正文
2. ✅ **AI生成自然评论** - 可自定义语气、风格
3. ✅ **链接嵌入** - 看起来自然
4. ✅ **批量生成** - 一次生成多条评论
5. ✅ **隐私保护** - 数据存储在本地

### 实施细则

#### 安装

1. 访问 Chrome Web Store 搜索 "AutoCommentAI"
2. 安装扩展
3. 配置AI模型（可能需要API key）

#### 使用流程

```
第1步：打开博客文章或论坛帖子
第2步：点击扩展图标
第3步：设置评论参数
  - Persona（角色）：专家/普通用户/热心粉丝
  - Tone（语气）：正式/友好/幽默
  - Style（风格）：简短/详细
第4步：插入你的网站链接
第5步：点击"生成评论"
第6步：审核并修改（重要！）
第7步：一键填充到评论框
第8步：手动提交（避免完全自动）
```

### 最佳实践

- ⚠️ **必须人工审核每条评论** - 不要盲目提交
- ⚠️ 每天不超过10-15条评论
- ⚠️ 在不同网站分散操作
- ⚠️ 不要在所有评论中都插入链接
- ⚠️ 优先高质量平台（DA 30+）

### 适用场景

- Medium文章评论
- Dev.to技术讨论
- Quora回答
- Reddit帖子（⚠️ 极度谨慎）
- 行业博客评论

### 资源链接

- **官方文档：** https://autocommentai.cc/docs/user-guide

## 八、关键注意事项
