## 🧪 本地测试

### 步骤1：在Chrome中加载扩展

1. **打开Chrome扩展页面**
   ```
   chrome://extensions/
   ```

2. **启用开发者模式**
   - 切换右上角的开关

3. **加载已解压的扩展程序**
   - 点击"加载已解压的扩展程序"
   - 选择克隆后的仓库根目录 `chrome-cagr`
   - 扩展应出现在列表中

4. **固定扩展**
   - 点击Chrome工具栏中的拼图图标
   - 固定"Smart CAGR Calculator"以便快速访问

### 步骤2：基本功能测试

1. **打开扩展**
   - 点击工具栏中的扩展图标
   - 弹窗应打开（400px × 600px）

2. **测试输入**
   - 在任意3个字段中输入值
   - 状态对号（✓）应出现
   - 模式指示器应自动更新

3. **测试计算**
   - 点击"立即计算"
   - 结果面板应出现
   - 验证计算是否正确

4. **测试清空**
   - 点击"清空"按钮
   - 所有输入应重置
   - 结果应隐藏

### 步骤3：多语言测试

1. **更改浏览器语言**
   ```
   chrome://settings/languages
   ```

2. **重新加载扩展**
   - 访问 `chrome://extensions/`
   - 点击扩展卡片上的重新加载按钮

3. **验证翻译**
   - 打开扩展
   - 检查所有文本是否已翻译
   - 测试所有9种语言

### 步骤4：离线测试

1. **禁用互联网**
   - 关闭Wi-Fi或断开网络

2. **测试扩展**
   - 打开扩展
   - 执行计算
   - 验证一切正常工作

3. **网站链接**
   - Logo和页脚链接应在新标签页打开
   - 浏览器将显示离线页面（预期行为）

## 🐛 调试

### Chrome开发者工具

1. **打开弹窗DevTools**
   - 右键点击扩展图标 → "检查弹出内容"
   - 或：打开弹窗 → 右键点击 → "检查"

2. **控制台日志**
   ```javascript
   // 检查初始化
   console.log('Smart CAGR Calculator initialized ✓');

   // 检查错误
   // 任何错误将以红色显示
   ```

3. **网络标签**
   - 应为空（无外部请求）
   - 扩展完全离线

### 常见问题与解决方案

#### 问题：扩展无法加载
```
错误：无法加载扩展
解决方案：
1. 检查manifest.json语法
2. 验证所有文件路径存在
3. 确保没有循环依赖
```

#### 问题：i18n不工作
```
错误：getMessage不是一个函数
解决方案：
1. 检查_locales文件夹结构
2. 验证messages.json格式
3. 确保manifest中有default_locale
```

#### 问题：计算错误
```
错误：结果中出现NaN或Infinity
解决方案：
1. 检查popup.js中的输入解析
2. 验证calculator.js中的公式
3. 测试边界情况（0、负数、巨大数字）
```

#### 问题：弹窗不打开
```
错误：点击时没有弹窗出现
解决方案：
1. 检查manifest中的popup.html路径
2. 验证HTML文件没有语法错误
3. 检查浏览器控制台错误
```
