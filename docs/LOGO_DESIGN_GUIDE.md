## 💡 设计理念

### 品牌核心价值

```
📈 增长 (Growth)
   → 视觉元素：上升曲线、箭头
   → 象征投资回报和业务增长

🧮 计算 (Calculation)
   → 视觉元素：数据点、百分比符号
   → 象征精确和专业

💼 金融 (Finance)
   → 颜色：蓝色（信任）、绿色（财富）
   → 象征可靠和专业

⚡ 智能 (Smart)
   → 设计风格：现代、简洁
   → 象征创新和高效
```

---

### 设计原则

1. **简洁性** (Simplicity)
   - 清晰的图形，易于识别
   - 避免过度装饰
   - 在小尺寸下依然清晰

2. **专业性** (Professionalism)
   - 使用金融行业常见的蓝色和绿色
   - 几何图形体现数学精确性
   - 现代的设计风格

3. **可识别性** (Recognizability)
   - 独特的增长曲线元素
   - 一致的视觉语言
   - 品牌色彩系统

4. **可扩展性** (Scalability)
   - SVG 矢量格式，无损缩放
   - 适配不同尺寸和场景
   - 响应式设计

## 📖 使用指南

### 在 React 组件中使用

我们提供了 `Logo` 组件，让您轻松使用不同版本的 LOGO：

#### 基础用法

```tsx
import { Logo } from '@/components/Logo'

// 完整版（默认）
<Logo />

// 或明确指定
<Logo variant="full" />
```

#### 使用不同方案

```tsx
// 方案 1：增长曲线徽章
<Logo variant="icon-1" />

// 方案 2：字母 C + 箭头
<Logo variant="icon-2" />

// 方案 3：几何风格
<Logo variant="icon-3" />

// Favicon 版本
<Logo variant="favicon" />
```

#### 自定义尺寸

```tsx
// 自定义宽高
<Logo variant="full" width={300} height={60} />

// 小尺寸图标
<Logo variant="icon-1" width={100} height={100} />

// 超大展示
<Logo variant="icon-3" width={400} height={400} />
```

#### 添加样式

```tsx
// 使用 className
<Logo
  variant="full"
  className="hover:opacity-80 transition-opacity cursor-pointer"
/>

// 响应式尺寸
<Logo
  variant="icon-1"
  className="w-20 h-20 md:w-32 md:h-32"
/>
```

---

### 直接使用 SVG 文件

如果您需要在非 React 环境中使用：

#### HTML

```html
<!-- 完整版 -->
<img src="/logo-full.svg" alt="CAGR Calculator" width="240" height="50">

<!-- 图标版本 -->
<img src="/logo-variant-1.svg" alt="CAGR Calculator" width="200" height="200">
```

### Q3: 如何修改 LOGO 颜色？

**A**: SVG 文件可以轻松修改颜色

**方法 1: 直接编辑 SVG 文件**

打开 `public/logo-variant-1.svg`，查找并替换颜色值：

```svg
<!-- 原来的蓝色 -->
<circle stroke="#3B82F6" />

<!-- 改为红色 -->
<circle stroke="#EF4444" />
```

**方法 2: CSS 修改**

```tsx
<Logo
  variant="icon-1"
  className="[&_path]:stroke-red-500 [&_circle]:stroke-red-500"
/>
```

**方法 3: 创建新的颜色变体**

复制 SVG 文件，创建新版本：
```bash
cp public/logo-variant-1.svg public/logo-variant-1-red.svg
# 然后编辑颜色
```

**A**: 可以，有多种方法

**方法 1: 在线转换工具**
- https://cloudconvert.com/svg-to-png
- https://svgtopng.com/

**方法 2: 使用设计软件**
- Figma: 导入 SVG → 导出 PNG
- Adobe Illustrator: 打开 SVG → 导出 PNG
- Inkscape（免费）: 打开 SVG → 导出 PNG

**方法 3: 命令行工具（ImageMagick）**
```bash
convert -background none -size 1024x1024 logo-variant-1.svg logo-variant-1.png
```

**方法 4: Node.js（sharp）**
```javascript
const sharp = require('sharp')
sharp('logo-variant-1.svg')
  .resize(1024, 1024)
  .png()
  .toFile('logo-variant-1.png')
```

### Q7: 如何在不同设备上显示不同尺寸的 LOGO？

**A**: 使用响应式设计

**Tailwind CSS 方式**:
```tsx
<Logo
  variant="full"
  className="w-32 h-8 md:w-40 md:h-10 lg:w-48 lg:h-12"
/>
```

**媒体查询方式**:
```css
.logo {
  width: 120px;
  height: 30px;
}

@media (min-width: 768px) {
  .logo {
    width: 180px;
    height: 40px;
  }
}

@media (min-width: 1024px) {
  .logo {
    width: 240px;
    height: 50px;
  }
}
```

**条件渲染**（移动端显示图标，桌面端显示完整版）:
```tsx
<>
  {/* 移动端 */}
  <Logo variant="favicon" className="md:hidden" width={40} height={40} />

  {/* 桌面端 */}
  <Logo variant="full" className="hidden md:block" width={180} height={40} />
</>
```

### Q8: LOGO 可以添加动画吗？

**A**: 可以！SVG 支持 CSS 动画和 SMIL 动画

**CSS 动画示例**:

```tsx
<Logo variant="icon-1" className="animate-bounce" />
```

**自定义动画**:
```css
@keyframes grow {
  0%, 100% { transform: scale(1); }
  50% { transform: scale(1.1); }
}

.logo-animate {
  animation: grow 2s ease-in-out infinite;
}
```

**SVG 内部动画**（编辑 SVG 文件）:
```svg
<circle r="20">
  <animate
    attributeName="r"
    from="20"
    to="25"
    dur="1s"
    repeatCount="indefinite"
  />
</circle>
```
