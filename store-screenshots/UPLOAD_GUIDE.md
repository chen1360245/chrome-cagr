## 🚨 重要注意事项

### 1. 上传顺序很重要！

**一定要按照文件名顺序上传：**
```
01 → 02 → 03 → 04 → 05
```

Chrome Web Store会按照您上传的顺序显示截图，第1张截图最重要！

### 2. 第1张截图的重要性

- 显示在商店搜索结果列表中
- 用户第一眼看到的截图
- 直接影响点击率和安装率
- 应该展示最有价值的功能（计算结果）

### 3. 语言策略

**推荐：使用Global Screenshots（全球截图）**
- 前4张英文（面向全球用户）
- 第5张中文（展示多语言能力）
- 所有用户看到相同的截图组合

**可选：使用Localized Screenshots（本地化截图）**
- 为不同语言区域提供不同截图
- 需要更多维护工作
- 适合有大量本地化内容的扩展

### 4. 文件命名规范

当前命名已优化：
- ✅ 数字前缀（01-05）确保正确排序
- ✅ 描述性名称（一眼看出内容）
