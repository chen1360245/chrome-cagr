# 占位图标

此文件夹需要3个PNG图标文件：
- icon16.png（16x16像素）
- icon48.png（48x48像素）
- icon128.png（128x128像素）

## 临时方案

在您生成正式PNG图标之前，您可以：

1. **使用SVG**：使用在线工具将 `icon.svg` 转换为PNG
2. **快速测试**：创建简单的彩色方块作为临时图标：
   - 打开任何图像编辑器（画图、预览等）
   - 创建16x16、48x48、128x128的绿色方块
   - 保存为icon16.png、icon48.png、icon128.png

3. **在线图标生成器**：
   - 访问：https://favicon.io/favicon-converter/
   - 上传icon.svg
   - 下载PNG包

## 重要说明

- 即使没有自定义图标，扩展也能正常工作
- Chrome将显示默认的拼图图标
- 建议使用自定义图标以获得专业外观
- 详细指南请查看ICON_INSTRUCTIONS.md
